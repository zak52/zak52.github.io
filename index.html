    Planetary Mapping with STAC Catalog


    This project focuses on the expansion of a Leaflet Map Viewer (CartoCosmos) which provides a modular map view of our planets.

    Our expansion focuses on displaying individual planetary images footprints based on the images location on a particular planet. These images are provided from SpatioTemporal Asset Catalogs (STAC).

    The next scope of this project is to provide the ability for the user to explore, click, and load the selected images within the Leaflet Viewer. This will provide the user the ability to pull and use large TIFF format images without the need to download or process the images.

    This will be beneficial to the planetary science community to focus more on their scientific research of our planets.

    AI Maze Runner


    The project, called "Maze Runner AI," takes in a picture of a 2D maze as input. The maze can have any size and can have any number of starting and ending points. The project uses different AI techniques to solve the maze, such as pathfinding algorithms like A* and Dijkstra's algorithm, and machine learning techniques like neural networks and random forests.

    Once the maze is given as input, the project uses these AI techniques to find the shortest path from the starting point to the ending point. The solution is then drawn on top of the original maze, highlighting the path that was found by the AI algorithms.

    The project also allows users to compare the performance of different AI techniques on the same maze, so they can see which technique works best for solving the maze. The project is written in Python and uses popular libraries like TensorFlow and NumPy for implementing the AI algorithms.

    Overall, the "Maze Runner AI" project provides a fun and interactive way to learn about different AI techniques and how they can be used to solve challenging problems like mazes.

    OS Simulator Project

    This is a simulation of a basic operating system, developed using C programming language. The project aims to demonstrate the functioning of an operating system and its various components, such as processes, memory management, and file system.

    Features

    • Process management: The simulator includes a process scheduler that simulates the execution of processes and implements various scheduling algorithms, such as FCFS (First-Come-First-Serve), SJF (Shortest-Job-First), and Round-Robin.
    • Memory management: The simulator includes a memory manager that simulates the allocation and deallocation of memory to processes, using various allocation strategies such as First-Fit, Best-Fit, and Worst-Fit.
    • File system: The simulator includes a file system that simulates the creation, deletion, and modification of files, as well as the allocation and deallocation of disk blocks.
